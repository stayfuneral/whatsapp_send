Whatsapp Send

отправка сообщений в Whatsapp

Если вы пользуетесь приложением Whatsapp, то наверняка успели заметить, что из приложения можно отправить сообщение только тем пользователям, которые есть в вашей адресной книге

Согласитесь - это крайне неудобно, ведь для единичного взаимодействия приходится добавлять номер телефона в свой контакт-лист.

С помощью нашего сервиса вы сможете отправить сообщение любому пользователю Whatsapp

Для этого введите нужный номер телефона в соответствующее поле. По желанию вы также можете добавить сообщение.

После нажатия на кнопку "Отправить" вы будете переадресованы в приложение или сервис whatsapp web, в зависимости вашего устройства

Введите номер телефона и сообщение, затем нажмите на кнопку "Отправить"
